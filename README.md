# Study

Репозиторий содержит учебные работы, ноутбуки с участием в соревнованиях на Kaggle и хакатонах.

## Содержание

### `ml/`
- `ML_experiment.ipynb` — ноутбук с практикой при прочтении хендбука по ML от Яндекса
- `wb_timeseries.ipynb` — решение для хакатона WildHack (индивидуальный трек).  
  *Результат: 53 место (не хватило времени на активную работу)*
- `neural_network_learn_ff/` — решения задач по курсу нейронных сетей на физфаке МГУ

### `algorithms/`
- `main.cpp` — реализации алгоритмов и структур данных с курса VK Education  
  *(основной фокус на структурах данных, все же для алгоритмов — LeetCode)*

## Примечание

В коде могут встречаться ошибки — это часть процесса обучения, поэтому не судите строго.
